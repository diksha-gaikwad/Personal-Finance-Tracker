# Personal-Finance-Tracker
Personal Finance Tracker is a Python-based console application that helps users track daily expenses, categorize spending, store data in JSON files, and generate basic reports. The project demonstrates file handling, modular programming, data validation, and error handling using real-world use cases.
Features

Add and manage daily expenses

Categorize expenses (travel, food, bills, etc.)

Save and load data using JSON

Backup expense data

Simple and user-friendly terminal menu

Modular and scalable project structure
